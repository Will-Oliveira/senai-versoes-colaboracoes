Principais comandos git, utilizados para gerenciar versionamento:<br>

$ cd "nome da pasta"

É possível escolher as pastas que terá acesso através deste comando.


$ git init

Inicia a git local na sua maquina local.


$ git status

Verifica a situação do seu arquivo na pasta de versionamento.


$ git add "nome do arquivo"

Incluirá nominalmente o arquivo descrito para a pasta de versionameto git.


$ git add .

Incluirá todos os arquivos que estiverm na pasta selecionada.


$ git commit -m "sua mensagem"

Insere um comentário personalizado acerca do arquivo.


$ git branch "nome do branch"

Cria uma ramificação para ser usada no código, sem alterar todo o projeto.


$ git checkout "nome do branch"

Alterna entre as ramificações das versões do projeto.


$ git push origin "nome da branch"

Usado para "subir" a alteração para o GitHub.
